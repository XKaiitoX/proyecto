GML Code. El GameMaker Language (también llamado simplemente GML) es el lenguaje de scripting propio de GameMaker.

Este lenguaje está estructurado para permitir a los usuarios crear sus juegos de forma intuitiva y flexible,

al tiempo que ofrece toda la potencia de cualquier otro lenguaje de programación importante.

Consiste en el uso de bloques de acción para construir la lógica del juego de una manera visual e intuitiva,

y es una herramienta ideal para las personas que están aprendiendo a programar,

o para las personas que tienen una inclinación más visual.
